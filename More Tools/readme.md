## 導入方法

### Windows

Behavior Pack を

```
com.mojang/behavior_packs/
```

へコピーします。

Resource Pack を

```
com.mojang/resource_packs/
```

へコピーします。

---

## ワールドへの適用

1. ワールド作成
2. リソースパックを有効化
3. ビヘイビアパックを有効化
4. 必要に応じて実験機能をON
5. ワールド開始

---

## コマンド

### アメジストの剣

```mcfunction
/give @s star:amethyst_sword
```

### エメラルドのツルハシ

```mcfunction

```

### クォーツの斧

```mcfunction

```

### 石炭のシャベル

```mcfunction

```

### ラピスラズリのクワ

